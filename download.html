melonJS uses part of, or a derivated version of the following libraries :

- MinPubSub: a micro publish/subscribe messaging framework (MIT License)
- Tween JS: a javaScript tweening engine for easy animations (MIT License)
- Jay Inheritance: a super fast prototypal inheritance microlib for the modern web (MIT License)
- Howler: a javascript audio library for the modern web (MIT License)
- SAT.js: a javascript Separating Axis Theorem implementation library by Jim Riecken (MIT License)
- quadtree-js: a QuadTree implementation in JavaScript by Timo Hausmann (MIT License)
