<static> registerPointerEvent(eventType, rect, callback, floating)

allows registration of event listeners on the object target.
melonJS defines the additional `gameX` and `gameY` properties when passing the Event object to the defined callback (see below)
Parameters:
Name Type Argument Description
eventType String The event type for which the object is registering
melonJS currently support ['pointermove','pointerdown','pointerup','mousewheel']
rect me.Rect object target (or corresponding region defined through me.Rect)
callback function methods to be called when the event occurs.
floating Boolean <optional>
 specify if the object is a floating object (if yes, screen coordinates are used, if not mouse/touch coordinates will be converted to world coordinates)
